Основы CSS

CSS управляет визуальным представлением HTML-элементов. Ключевые понятия:

Selectors

CSS uses selectors to target elements:

  • h1 - Element selector
  • .class - Class selector
  • #id - ID selector

Properties

Common CSS properties:

  • color - Text color
  • background-color
  • font-size
  • margin/padding
